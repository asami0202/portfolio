Case Study 01 — Restaurant Industry
Automated Reservation System for a Small Restaurant
Google Apps Script · Google Sheets · Calendar

The Situation

  • Reservations scattered across phone, email, and walk-ins
  • Phone calls unanswered during busy service hours
  • Email reservations required multiple back-and-forth messages
  • Missed bookings and human errors were common
  • No budget for expensive DX software
  • Staff had limited tech experience

What I Did First

  • Mapped every type of reservation and its decision point
  • Identified which cases could be automated vs. needed human judgment
  • Defined rules for peak seasons, holidays, and large groups
  • Decided: not everything should be automated

Key design principle: Clear separation between what the system handles automatically and what gets escalated to a human.

What Was Built

  • 24/7 online booking form
  • Automated availability checks (time slots, capacity, holidays)
  • Out-of-scope requests redirected to email inquiry
  • Instant confirmation emails sent automatically
  • All bookings consolidated in one Google Sheet
  • Auto-sync to Google Calendar for owner visibility
  • One-click calendar entry for phone reservations
  • Data structured for future analysis (peak hours, guest type)
✓
Dramatically fewer missed reservations
✓
Phone pressure during service hours eliminated
✓
No more single-person dependency
✓
Owner has real-time operational visibility
✓
Staff can run the system without IT help
Case Study 02 — Financial Industry
KPI Report Automation: From Hours to Minutes
Power Automate Desktop · Excel · Power Query

The Situation

  • Weekly and monthly KPI reports took hours to compile
  • Data pulled manually from multiple systems
  • Copy-paste errors and calculation mistakes were regular risks
  • Only one person knew how to do it — creating a bottleneck

What I Did First

  • Audited every data source and where it lived
  • Identified which metrics were truly necessary
  • Determined which steps could be fully automated

Key decision: All standardized processing steps would be fully automated — no exceptions.

✓
~85% reduction in reporting time
✓
Zero data entry errors
✓
Anyone can run the process — no specialist required
✓
Team can now focus on actual analysis, not data wrangling
